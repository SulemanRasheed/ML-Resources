<h1 id="open-source-resources-to-learn-machine-learning-deep-learning">Open Source Resources to Learn Machine Learning/ Deep Learning</h1>
<h2 id="mindset-required">Mindset Required</h2>
<ol>
<li>
<p><a href="https://www.fast.ai/2016/10/08/teaching-philosophy/">Follow Top Down Approach</a> After learning a few basics (I really mean only a few!), go for actually making things. Do your own projects. Learn while you make mistakes. Initially try to use high level libraries to get things done and then go for implementation details.<br>
check this talk by Rachel Thomas on  <a href="https://vimeo.com/214233053">How to Learn Deep Learning (when you’re not a computer science PhD)</a></p>
</li>
<li>
<p>Avoid <a href="https://www.youtube.com/watch?v=MxxAc9a4ILU">Tutorial Purgatory</a>, Apply what you have learned and only learn new stuff when needed by the project</p>
</li>
<li>
<p>Manage your time like a professional software developer, use <a href="https://www.youtube.com/watch?v=UdvwVYAaWEc">kanbanflow</a> to schedule your week</p>
</li>
<li>
<p>Market yourself, no one else is gonna do it for you! Focus on making personal projects and make a <a href="https://machinelearningmastery.com/build-a-machine-learning-portfolio/">machine learning portfolie</a> to show others what you have done.</p>
</li>
</ol>

<h2 id="ml-courses">ML Courses</h2>
<ol>
<li><a href="https://www.fast.ai/">Fast ai Course for Deep Learning</a> Good for Beginners</li>

</ol>
<p>Andrew’s courses are good for general understand and mathematical intuitions. From Programming perspective, fast ai is best course for beginners in deep learning. Andreas’s course is master’s level course in ML so it assumes some background knowledge but it is really worth giving a try.</p>
